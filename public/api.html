API JSON - Cinematography

La siguiente API proporciona información sobre películas.

Rutas para películas

Obtener todas las películas

GET /api/films/

Devuelve un JSON con todas las películas. Los campos son:

  • id: identificador único de la película.
  • name: nombre de la película.
  • protagonist: id del protagonista
  • adaptation: si se trata de una adaptación.
  • date: fecha de estreno de la película.
  • country: país de producción de la película.
  • director: nombre del director.
  • img: URL de la imagen de la película.

Ejemplo de respuesta:

{ "films": [ { "id": 1, "name": "The Godfather", "protagonist": "Marlon Brando", "adaptation": null, "date": "1972-03-24", "country": "USA", "director": "Francis Ford Coppola", "img": "https://www.imdb.com/title/tt0068646/mediaviewer/rm599965184/" }, { "id": 2, "name": "Forrest Gump", "protagonist": "Tom Hanks", "adaptation": null, "date": "1994-06-23", "country": "USA", "director": "Robert Zemeckis", "img": "https://www.imdb.com/title/tt0109830/mediaviewer/rm1748727040/" } ] }

Obtener una película por id

GET /api/films/:id

Devuelve un JSON con la película cuyo identificador es :id.

Ejemplo:

{ "films": [ { "id": 1, "name": "The Godfather", "protagonist": "Marlon Brando", "adaptation": null, "date": "1972-03-24", "country": "USA", "director": "Francis Ford Coppola", "img": "https://www.imdb.com/title/tt0068646/mediaviewer/rm599965184/" } ] }

Insertar una película

POST /api/films/

Crea una nueva película con los datos proporcionados del formulario, y devuelve un JSON con los datos de la película

Eliminar una película

app.delete('/api/films/:id'

Esta ruta permite eliminar una pelicula tomando el id como parámetro y la borra tambien de la base de datos.

Actualizar una película

POST /api/films/:id

Esta ruta permite actualizar la información de una pelicula por su ID como parámetro tambien actualiza la película en la base de datos con los nuevos datos.

Rutas para actores

Obtener todos los actores

GET /api/actors/

Devuelve un JSON con todos los actores. Los campos son:

  • id: identificador único del actor.
  • name: nombre del actor.
  • cast: id de su pelicula.
  • birth_date: fecha de nacimiento.
  • year_of_death: fecha de fallecimiento (si ha muerto).
  • nationality: nacionalidad del actor.
  • years_active: años en activo del actor.
  • img: URL de la imagen del actor.

    • Ruta para obtener el actor y su pelicula

    GET /api/actors/all

    Esta ruta devuelve la información los actores, incluyendo la película en la que actuaron.

    Obtener un actor por id

    GET /api/actors/:id

    Devuelve un JSON con el actor cuyo identificador es id.

    GET /api/actors/:id

    Esta ruta permite obtener información sobre un actor en particular tomando el ID del actor como parámetro

    Insertar una película por id

    POST /api/actors/

    Crea un nuevo actor con los datos proporcionados del formulario, y devuelve un JSON con los datos del actor

    Eliminar una actor

    app.delete('/api/actor/:id'

    Esta ruta permite eliminar un actor tomando el id como parámetro y la borra tambien de la base de datos.

    Actualizar actores

    POST /api/actors/:id

    Esta ruta permite actualizar la información de un actor tomando el su ID como parámetro. Actualiza el actor correspondiente en la base de datos con los nuevos datos.
